To create a presentation with preview, the images must have been created beforehand using ImageResize with the settings shown below. This will create two .imgtags files, one of which must be specified in the presentation dialog.

  • All images are in the same directory
  • Two image sizes
    1. a small one (e.g. 120px) for the thumbnail
    2. a large one (e.g. 1920) for the full view
  • Custom renaming with this template:
    img${INDEX.ifmt(1,3)}_${SIZENAME}.${FILEEXT}
  • These two size names:
    1. THUMBNAIL
    2. SCREEN
